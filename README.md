## 搭建网站完整步骤

老高的方案核心分这几步，我按现在搬瓦工的实际情况更新了细节：

### 第一步：购买 VPS 并获取 SSH 登录信息

1. 打开 [搬瓦工官网](https://bit.ly/BanWaGon)，注册账号，选择套餐和付款周期
2. 付款完成后，进入后台 **Services → My Services**，找到购买的 VPS
3. 点击 **KiwiVM Control Panel** 进入 KiwiVM 控制面板
4. 左侧菜单找 **Root password modification**，重置 root 密码（只显示一次，记好）
5. 在 KiwiVM 首页记下你的 **IP 地址**和 **SSH 端口**（不是 22，搬瓦工分配的是随机端口）

注意这里很多人卡住：SSH 端口是在 KiwiVM 面板里看的，不是默认的 22。用 PuTTY 或 Xshell 连接时，服务器地址填 IP，端口填面板里显示的那个数字。

### 第二步：安装操作系统

KiwiVM 面板左侧有 **Install new OS**，选 Ubuntu 22.04 或 Debian 12，点安装。安装过程几分钟，完成后重新连接 SSH。

顺便说：搬瓦工已经支持 Ubuntu 26.04，你在 KiwiVM 面板里可以一键重装，新版本没什么坑。

### 第三步：安装 Web 环境（三种方案选一）

**方案一：宝塔面板（最简单，推荐新手）**

宝塔是可视化的 Web 面板，装好之后所有操作都有图形界面，不用敲命令。SSH 进 VPS 后执行宝塔的安装命令，装完后按提示在浏览器打开面板地址，安装 LNMP 或 LAMP 套件，几分钟完成。

**方案二：LNMP 一键包（小内存推荐）**

LNMP = Linux + Nginx + MySQL + PHP。Nginx 比 Apache 占内存少，1GB 内存的 VPS 推荐用这套。SSH 连进去运行 LNMP 安装命令，选好 MySQL 和 PHP 版本等待完成。

**方案三：LAMP 一键包**

LAMP = Linux + Apache + MySQL + PHP。老高早期用的是这套，兼容性好，内存允许的情况下也完全没问题。

三种方案选一就行。预算方案（$49.99/年 那个）1GB 内存建议选 LNMP 或宝塔。

### 第四步：域名解析

把你的域名 A 记录指向搬瓦工 VPS 的 IP 地址。解析生效通常 5-30 分钟。生效后 ping 你的域名能返回 VPS 的 IP，说明解析成功了。

### 第五步：新建网站并安装 WordPress

1. 在宝塔或 LNMP 面板里添加新网站，填入你的域名，数据库信息同步创建
2. 下载 WordPress 安装包，通过 FTP（FileZilla）或宝塔文件管理上传到网站根目录
3. 解压文件，给目录赋权：`chown -R www:www /你的网站目录`
4. 浏览器访问你的域名，按 WordPress 安装引导填入数据库信息
5. 五分钟内完成安装，前台是你的网站，后台在 `你的域名/wp-admin`

安装完成。不夸张，走完这五步，一个能跑的 WordPress 网站就在线了。

## 几个容易踩的坑

**坑一：SSH 连不上**

八成是端口填错了。搬瓦工的 SSH 端口在 KiwiVM 面板首页显示，不是 22。

**坑二：数据库密码忘了**

宝塔面板里可以直接改，LNMP 的话用 `mysqladmin` 命令重置。老高的教程里这段有详细说。

**坑三：IP 被封**

在 KiwiVM 面板里付费换一个 IP，目前费用大约 $8.79。如果是 CN2 GIA-E 套餐，也可以先换机房（换了机房 IP 会变），看看新 IP 能不能用。

**坑四：内存不够跑 WordPress**

1GB 内存是够的，但如果装了太多插件或者没做缓存，TTFB 会很慢。宝塔里安装 WP Rocket 或 LiteSpeed Cache，开启页面缓存，加载时间能压下来不少。

## FAQ

**Q：搬瓦工搭建网站需要备案吗？**

不需要。搬瓦工的服务器在境外（美国、香港、日本等），境外服务器建站不需要国内 ICP 备案。但如果你的网站面向国内用户，不备案在国内访问有可能被屏蔽。想做中文内容站且主要面向国内，建议考虑备案，或者直接用香港机房（物理在香港也不需要大陆备案）。

**Q：1GB 内存够不够跑 WordPress？**

够。跑一个 WordPress 博客，1GB 内存加上正常的缓存配置，日常几千 PV 完全没问题。如果流量大了，可以升级到 CN2 GIA-E 的 2GB 方案，搬瓦工后台可以直接在线升级套餐。

**Q：老高的建站教程现在还适用吗？**

框架适用。核心步骤——买 VPS、装环境、配域名、上 WordPress——这个流程没变。变的是他教程里提到的某些控制面板（比如 Kangle、EasyPanel）比较老了，现在更推荐宝塔面板，界面更现代，功能更全，新手更容易上手。

**Q：买了 VPS 之后多久可以开始建站？**

付款后几分钟内 VPS 就开通了，KiwiVM 面板里可以立刻看到状态。安装系统、部署环境、装 WordPress，整个流程熟练的话 1 小时内搞定，第一次操作的话留 2-3 小时。

**Q：搬瓦工和虚拟主机有什么区别？**

虚拟主机是共享资源，商家帮你管好了环境，你只管上传网站文件，价格便宜但定制空间小、资源受限。搬瓦工 VPS 是独立的虚拟服务器，你有 root 权限，想装什么装什么，但自己负责所有运维。建站长期来看，VPS 灵活性高，扩展性更好。
